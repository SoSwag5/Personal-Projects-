JobPro Prototype - Project Overview

This project is a prototype of the JobPro website, created to demonstrate the core features and design elements of the system. The prototype includes static HTML files enhanced with JavaScript and CSS for interactivity and animations.

Features:
- Static HTML structure representing the JobPro website.
- JavaScript for adding dynamic content and interactivity.
- CSS animations for visual effects and smooth transitions.
- A responsive layout to ensure the site looks good on different devices.
